## Credibility red flag
A LinkedIn Pulse result still states **ETIAS = €7, launching 2025**. This is **stale/wrong**
— gov.uk confirms **€20, Q4 2026, 6-month grace to ~April 2027**. Rule: newsjacks cite
**tier-5 only** (gov.uk / eu-LISA / europa.eu / ABTA); treat Pulse "expert" posts as
angle inspiration, never as fact. This is exactly why signals.json is refreshed from
official sources each batch, not scraped from LinkedIn.
