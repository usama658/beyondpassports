# Team headshot prompts

One folder per team member, each with a `prompt.md` (realistic + professional image-generation prompt) and space to drop the generated photo. Full write-up + composition rules: `../team-profiles.md`.

**Rules:** generate each person separately (group prompts blend faces); keep "ordinary / average / not a model"; soft even light + neat grooming = professional; natural skin texture + slight asymmetry + phone-camera feel = real; skip the grime. Save each as ~800×800, 4:5 crop, into `public/assets/img/team/`.
