# LinkedIn KPIs — Beyond Passports

Owner reference. Beyond Passports is a **visa consultancy / lead-gen service**
(Schengen prep, tour packages, long-stay), not a personal-brand coaching offer —
so conversion is the north star and the "newsletter / discovery call" rows are
swapped for the real funnel (checklist / eligibility starts, WhatsApp enquiries).

## Profile vs Page (important)
- `linkedin.com/in/beyond-passports` is a **personal profile** (`/in/`), not a
  Company Page (`/company/`).
- Personal profiles get more organic reach (LinkedIn throttles Pages ~2-5×) — good
  now, but a faceless brand-as-person underperforms. Best: a **named human** (founder /
  case-lead) posts from a personal profile **and** a Company Page exists for trust.
- People engage with faces, not logos. If nobody fronts it, halve the reach targets.

## Attribution — do this first
Tag the profile's website link so LinkedIn clicks show in GA4 (site already runs GA4,
consent-gated):
```
https://beyondpassports.co.uk/?utm_source=linkedin&utm_medium=social&utm_campaign=profile
```
Read it back: GA4 → Acquisition → Traffic acquisition → filter `linkedin`.
