# Social media campaigns — Beyond Passports

Owner reference. Everything ties one buyer fear → one product. The whole edge is
cutting refusal risk. Compliance baked in (see bottom).

## Products
1. **Schengen prep** — check/prepare/submit, cut refusal risk, book appointment.
2. **Tour packages** — "book the trip, we clear the visa" (enquiry-only until ATOL).
3. **Long-stay** — Spain/Portugal/Germany move-to-Europe (prep + assist, not legal advice).

---

## Core campaigns (evergreen, always-on)

### 1. "Why visas get refused" — top of funnel
- Format: carousels / reels, one refusal reason each (weak ties, thin funds, wrong
  consulate, booked-before-visa). Source: refusal taxonomy + guides engine.
- Goal: reach + saves. CTA: free document checklist.
- Platforms: Instagram, TikTok, LinkedIn, Pinterest.

### 2. "Book the trip, we clear the visa" — tour packages
- Format: destination reels (Paris/Venice/Santorini) + "visa included" hook.
- Goal: enquiries. CTA: WhatsApp.
- Platforms: Instagram, TikTok, Facebook, Pinterest.
- ⚠️ ATOL gate — enquiry-only until licensed. No priced sale in ads.

### 3. "Refused before?" — high-intent / retargeting
- Format: "a rebuild, not a retry" — outcome-led, honest (no guarantee).
- Goal: convert the refused. CTA: WhatsApp.
- Platforms: Meta retargeting (Pixel live), LinkedIn.

### 4. "Move to Europe" — long-stay, authority-led
- Format: explainer carousels — Spain non-lucrative, Portugal D7, Germany Chancenkarte.
- Goal: leads + authority. CTA: eligibility check.
- Platforms: LinkedIn (best fit), YouTube, Reddit/Quora (answer, don't spam).

## Seasonal / burst
- **Jan–Mar "Summer visa now"** — beat peak appointment crunch; urgency: slots vanish.
- **Sep–Oct "Christmas market trips"** — tour packages.
- **Rule-change newsjack** — EES/ETIAS launches, Chancenkarte updates → big reach.

## Platform roles (don't cross-post identically)
| Platform | Role | Primary campaign |
|---|---|---|
| Instagram / TikTok | reach, fear-led reels | Refusals + Tours |
| LinkedIn | authority, long-stay, B2B | Move to Europe |
| Facebook | Meta ads + retargeting | Refused-before + Tours |
| Pinterest | evergreen checklist SEO | Refusals + destinations |
| YouTube | deep explainers | Long-stay + how-it-works |
| Reddit / Quora | answer real questions | organic trust, link sparingly |

## Funnel (every campaign)
Reach post → free tool (checklist / eligibility check) → WhatsApp enquiry → order.
Every link UTM'd (`utm_source=<platform>`) so GA4 attributes.

## Measurement
Per campaign: reach → tool starts → WhatsApp enquiries → orders.
North-star = enquiries, not likes.

## Compliance (baked in)
- No approval/visa guarantees (DMCCA). "Cut refusal risk" OK; "removes" not.
- Tours enquiry-only until ATOL + PTR in place.
- Long-stay = prep/assist, not legal advice (RDG etc.).
- No fabricated stats; numbers pull from SiteStats.
- Cookie/pixel consent already gated (UK PECR).
