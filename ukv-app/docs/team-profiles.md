# Team profiles — Beyond Passports

Four team members for the site team section, with realistic Gemini / Imagen / Midjourney headshot
prompts. Independent Schengen visa service (UK). Names illustrative — swap for real staff if used
publicly.

**Goal: ordinary, real, everyday people — not models — each clearly different.** For four believable
staff photos, vary FOUR things per person, not just hair: (1) a plain distinct face + small
imperfections, (2) a different **posture / camera angle**, (3) a different **background / setting**,
(4) different everyday lighting. All four are baked into the prompts below. Generate each person
separately.

## Realism checklist

- **Say "ordinary / average / plain / not a model"** in every prompt — biggest lever for normal faces.
- **Generate each person separately.** Group prompts blend faces + poses.
- **Everyday flat indoor light**, not studio glamour.
- **Vary posture, angle AND background per person** (table above) — same pose/background across all four is as much an AI tell as the same face.
- **One or two real imperfections each** — tired eyes, blemish, frizzy hair, gap tooth, jowls, receding hair.
- **4:5 portrait, slightly off-centre, slightly soft focus** — like a real phone photo.
- Negatives if supported: `--no model, glamour, beauty filter, airbrushed, plastic skin, symmetrical face, studio glam lighting`.
